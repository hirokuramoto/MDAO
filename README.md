# MDAO

MDAO is a connection tool for OpenFOAM and OpenMDAO.


# Requirement

* python-3.6.9
* openmdao 3.1.1
* numpy
* pandas
* matplotlib



# Installation

Requirementで列挙したライブラリなどのインストール方法を説明する

```bash
pip install huga_package
```

# Usage

DEMOの実行方法など、"hoge"の基本的な使い方を説明する

```bash
git clone https://github.com/hoge/~
cd examples
python demo.py
```
